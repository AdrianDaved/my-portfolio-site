# 🌐 My Portfolio Site

My personal portfolio website built to showcase projects, skills, and experiences in IT and Cybersecurity. 🔐

## 🛠️ Tech Stack

- ⚡ **Framework:** [Next.js](https://nextjs.org/) 16
- 📝 **Language:** TypeScript
- 🎨 **Styling:** Tailwind CSS 4
- ⚛️ **UI Library:** React 19

## 📁 Project Structure

```
app/
├── components/   # 🧩 Reusable UI components
├── globals.css   # 🎨 Global styles
├── layout.tsx    # 📐 Root layout
└── page.tsx      # 🏠 Home page
```
